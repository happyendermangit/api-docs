# Token

`GET` `/miscs/token`

Generate's a random token **for fun!**

This is ONLY for fun, the token's that are generated ARE FAKE and NOT REAL
:::

## Response:

| Field | Type | Description|
|-------|------|------------|
| token | string | The generated token (FAKE TOKEN!!!) |
